# Microservices-springboot
A beginner-friendly to advanced implementation of microservices architecture using Spring Boot and Spring Cloud, starting from basics and gradually introducing service discovery, API gateway, and inter-service communication.

## 📌 Overview
This repository is designed to learn and implement microservices architecture from scratch using Spring Boot. It starts with basic REST services and gradually evolves into a complete microservices system.

---

## 🎯 Learning Goals
- Understand monolith vs microservices
- Build independent services
- Implement service discovery
- Use API Gateway
- Handle inter-service communication
